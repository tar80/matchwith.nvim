VARIABLES                                                   *matchwith-variable*

g:matchwith_disable                                        *g:matchwith_disable*
                If truthy, matchwith is globally disabled.
                Default: nil

b:matchwith_disable                                        *b:matchwith_disable*
                If truthy, matchwith is disabled for the current buffer.
                Default: nil

g:matchwith_captures                                      *g:matchwith_captures*
                Treesitter captures to be used for matching.
                See |matchwith-captures-config| for details.

g:matchwith_debounce_time                            *g:matchwith_debounce_time*
                Wait time (ms) before highlighting after cursor movement.
                Default: 50

g:matchwith_depth_limit                                *g:matchwith_depth_limit*
                Maximum search depth in the syntax tree.
                Default: 20 (stored as 10 * 2)

g:matchwith_ignore_filetypes                      *g:matchwith_ignore_filetypes*
                List of filetypes to ignore.
                Default: { "vimdoc" }

g:matchwith_ignore_buftypes                        *g:matchwith_ignore_buftypes*
                List of buftypes to ignore.
                Default: { "nofile" }

g:matchwith_ignore_parsers                          *g:matchwith_ignore_parsers*
                List of Treesitter parsers to ignore.
                Default: { "markdown" }

g:matchwith_indicator                                    *g:matchwith_indicator*
                Time (ms) to display an indicator on markers when off-screen.
                Default: 0

g:matchwith_priority                                      *g:matchwith_priority*
                Highlight and sign priority.
                Default: 100

g:matchwith_show_next                                    *g:matchwith_show_next*
                Whether to highlight the next available match.
                Default: false

g:matchwith_show_parent                                *g:matchwith_show_parent*
                Whether to highlight the parent node match.
                Default: false

g:matchwith_sign                                              *g:matchwith_sign*
                Whether to show symbols in the sign column for off-screen
                matches.
                Default: false

g:matchwith_symbols                                        *g:matchwith_symbols*
                Table of symbols indicating off-screen directions.
                See |matchwith-symbols-config|.

g:matchwith_word_highlight                          *g:matchwith_word_highlight*
                Enable/disable word highlighting (word_highlight branch only).
                Default: false

g:matchwith_avoid_word_type                        *g:matchwith_avoid_word_type*
                Nodes excluded from word highlight (word_highlight branch only).

        g:matchwith_word_highlight                    |matchwith-word-highlight|

        g:matchwith_avod_word_type                   |matchwith-avoid-word-type|

CONFIGURATION                                          *matchwith-configuration*

captures                                             *matchwith-captures-config*

                Captures to apply matchwith. "*" is the fallback for all
                filetypes.

                        Type: `table`
                        Default: >lua
                       {
                         ['*'] = { 'tag.delimiter', 'punctuation.bracket' },
                         lua = { 'keyword.function', 'keyword.repeat',
                                 'keyword.conditional', 'punctuation.bracket',
                                 'constructor' },
                         vim = { 'keyword.function', 'keyword.repeat',
                                 'keyword.conditional', 'punctuation.bracket',
                                 'constructor', 'keyword.exception' },
                       }
<
debounce_time                                               *matchwith-debounce*

                Wait time before highlighting after cursor movement.

                        Type: `integer`
                        Default: 50

depth_limit                                              *matchwith-depth-limit*

                Specifies the maximum start depth for each match of the syntax
                tree. The actual limit value is twice the specified value.

                        Type: `integer`
                        Default: 20 (10 * 2)

ignore_filetypes                                    *matchwith-ignore-filetypes*

                Filetypes that do not perform matchwith.

                        Type: `table`
                        Default: { "vimdoc" }

ignore_buftypes                                      *matchwith-ignore-buftypes*

                Buftypes that do not perform matchwith.

                        Type: `table`
                        Default: { "nofile" }

ignore_parsers                                        *matchwith-ignore-parsers*

                Specify parsers to suppress reading.

                        Type: `table`
                        Default: { "markdown" }

jump_key                                                     *matchwith-jumpkey*

                Key to jump between matched pairs.

                        Type: `string`
                        Default: nil (e.g., set to "%")

indicator                                                  *matchwith-indicator*

                If match is off-screen. Number of milliseconds to display
                indicator on marker. For example, if the cursor has a box
                shape, the highlight of the bracket under the cursor is
                unknown, so an indicator is displayed.

                        Type: `integer`
                        Default: 0

priority                                                    *matchwith-priority*

                Marker and sign priority.

                        Type: `integer`
                        Default: 100

show_next                                                  *matchwith-show-next*

                Highlight the next capture match.

                        Type: `boolean`
                        Default: false

show_parent                                              *matchwith-show-parent*

                Highlight the parent node match.

                        Type: `boolean`
                        Default: false

sign                                                            *matchwith-sign*

                If match is off-screen. Display a symbol in signcolumn.

                        Type: `boolean`
                        Default: false

symbols                                               *matchwith-symbols-config*

                Directional symbols for off-screen matches.

                        Type: `table`
                        Default: >lua
                                { '↑', '↓', '→', '↗', '↘', '←', '↖', '↙' }
<
word_highlight                                        *matchwith-word-highlight*

                (This feature is specific to the word_highlight branch.)
                Enable/Disable word_highlight.

                        Type: `boolean`
                        Default: false

avoid-word-type                                      *matchwith-avoid-word-type*

                (This feature is specific to the word_highlight branch.)
                Specifies the treesitter node that is not subject to
                word_highlight.

                        Type: `table`
                        Default: >lua
                        {
                                'comment',
                                'string',
                                'codeblock',
                                'heading',
                                'delimiter'
                        }
<

word_highlight                                        *matchwith-word-highlight*

        (This feature is specific to the word_highlight branch.)
        Enable/Disable word_highlight.

            Type: `boolean`
            Default: false

avoid-word-type                                       *matchwith_avod_word_type*

        (This feature is specific to the word_highlight branch.)
        Specifies the treesitter node that is not subject to word_highlight.

            Type: `table`
            Default: {
                'comment',
                'string',
                'codeblock',
                'heading',
                'delimiter'
            }

MAPPING                                                      *matchwith-mapping*

The following <Plug> maps are available for operator-pending mode:

<Plug>(matchwith-operator-i)          Inner range of current/next matchpair.
<Plug>(matchwith-operator-a)          A range (inclusive) of current/next.

------------------------------------------------------------------------------
HLGROUP                                                      *matchwith-hlgroup*

*Matchwith*            Current matchpair highlight.
*MatchwithOut*         Current matchpair (off-screen).
*MatchwithNext*        Next matchpair highlight.
*MatchwithNextOut*     Next matchpair (off-screen).
*MatchwithParent*      Parent matchpair highlight.
*MatchwithParentOut*   Parent matchpair (off-screen).
*MatchwithSign*        Symbols in the sign column.
*MatchwithWord*        Word highlighting (word_highlight branch).

*@keyword.do.matchwith*
                Special highlight for Treesitter `do_statement` nodes.
                Override this if you wish to change specific language behavior
                via `matchwith/queries`.

*MatchwithWord*

        Used for word-highlight
        (Only valid in the word_highlight branch.)

==============================================================================
DEPRECATED                                                *matchwith-deprecated*

|alter_filetypes|

        This option allows you to specify an alternative parser name when
        the filetype and the actual parser name to be used differ.

|off_side|

        Specifies the file types to which off-side rules apply.
        Specify the capture group within |matchwith-captures|.
